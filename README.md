## WordLeaf
* A memo application that resembles a deck.
* 単語帳をイメージしたメモアプリ。


## Description
* You can register words and their meanings in series (or by note), and if necessary, you can register details.
* 単語とその意味をシリーズ（またはノート別に）登録し、必要があれば長文や画像などを記述できる詳細を登録できる。
